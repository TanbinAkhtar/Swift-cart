1. Global Scope
A variable declared outside any function or curly braces {} is in the Global Scope. These variables can be accessed from anywhere in your entire JavaScript file.
2. Function Scope
When you declare a variable inside a function (using var, let, or const), it is local to that function. You cannot access it from outside.
3. Block Scope
Block Scope means a variable is only available within the specific pair of curly braces {} where it was defined.
